LevelSet.UpdateLists(EltsChannel, EltsTipNew, FillFrac, levelSet, mesh)[source]¶

This function update the Element lists, given the element lists from the last time step. EltsTipNew list can have partially filled and fully filled elements. The function update lists accordingly.

Parameters:
  • EltsChannel (ndarray-int) – channel elements list
  • EltsTipNew (ndarray-int) – list of the new tip elements, including fully filled cells that were tip cells in the last time step
  • FillFrac (ndarray-float) – filling fraction of the new tip cells
  • levelSet (ndarray-float) – current level set
  • mesh (CartesianMesh object) – the mesh of the fracture
Returns:

new channel elements list (ndarray-int): new tip elements list (ndarray-int): new crack elements list (ndarray-int): new ribbon elements list (ndarray-int): list specifying the zero vertex of the tip cells. (can have value from 0 to 3,

where 0 signify bottom left, 1 signifying bottom right, 2 signifying top right and 3 signifying top left vertex)

(ndarray-int): specifies which region each element currently belongs to
Return type:

(ndarray-int)
LevelSet.reconstruct_front(dist, EltChannel, mesh)[source]¶

Track the fracture front, the length of the perpendicular drawn on the fracture and the angle inscribed by the perpendicular. The angle is calculated using the formulation given by Pierce and Detournay 2008.

Parameters:
  • dist (ndarray-float) – the signed distance of the cells from the fracture front
  • EltChannel (ndarray-int) – list of Channel elements
  • mesh (CartesianMesh object) – the mesh of the fracture
LevelSet.reconstruct_front_LS_gradient(dist, EltChannel, mesh)[source]¶

Track the fracture front, the length of the perpendicular drawn on the fracture and the angle inscribed by the perpendicular. The angle is calculated from the gradient of the level set.

Parameters:
  • dist (ndarray-float) – the signed distance of the cells from the fracture front
  • EltChannel (ndarray-int) – list of Channel elements
  • mesh (CartesianMesh object) – the mesh of the fracture
